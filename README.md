## Development Setup
1. Clone the repository
2. Install dependencies:
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```
3. Start development servers:
   ```bash
   # Terminal 1 - Client
   cd client && npm run dev
   
   # Terminal 2 - Server
   cd server && npm run dev
   ```
